# QuiQui Questions — NLP @ TH Nürnberg

Interactive lecture questions for the course *Introduction to Natural Language Processing* (Prof. Dr. Jens Albrecht, TH Nürnberg).

Used with [QuiQui](https://github.com/albrechtje/quiqui) — a lightweight live audience response tool. No app, no login required for students.

## Files

| File | Content |
|------|---------|
| `config.yaml` | Session URL and title |
| `NLP*.yaml` | Question files in QuiQui yaml notation |
